# 🚗 Parking Management System – Database Design

A structured and scalable database design for a **Parking Management System**, built to efficiently manage vehicles, parking spaces, sessions, and payments.

---

## 📌 Overview

This system is designed to handle:

* Vehicle entries & exits
* Parking slot allocation
* Zone-based parking management
* Ticket generation
* Payment tracking

It follows a **normalized relational design** to ensure scalability and avoid redundancy.
